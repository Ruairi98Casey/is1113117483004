Platform as a Service (PaaS) - Cloud 9

-Platform as a Service (PaaS), is a category of cloud computing that provides a platform and environment to allow developers to build applications and services over the internet. PaaS services are hosted in the cloud and accessed by users simply via their web browse

-Cloud9 is a company that provides you a development environment in Cloud for Web Applications. It is an online integrated development environment, published as open source which supports supports hundreds of programming languages. It allows developers to get started with coding immediately with pre-configured workspaces, collaborate with their peers with collective coding features, and web development features like live preview and browser compatibility testing.
